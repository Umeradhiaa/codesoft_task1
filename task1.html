Rock-Paper-Scissors

We have developed an interactive Rock-Paper-Scissors game leveraging Arduino technology and sensors. In this game, participants engage by presenting one of the symbols related to Rock, Paper, or Scissors, which is then detected by a sensor. The Arduino processes the input and generates an output displayed through lights, each representing a different symbol: red for Rock, yellow for Paper, and green for Scissors. The outcome of the game is determined based on the comparison between the participant's input and a randomly generated output, providing an entertaining experience this project seamlessly blends hardware and software for an engaging experience.
